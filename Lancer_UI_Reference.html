Glassmorphism recipe: background: rgba(255,255,255,0.03–0.06); border: 0.5px solid rgba(255,255,255,0.07–0.12); backdrop-filter: blur(10–16px); border-radius: 12–16px. Use consistently on all card components.

Background star animation

Every page has a dark #0D0D12 background with two canvas-based animations:
1. Twinkling stars — ~200 randomly placed dots, varying opacity, flickering with sine wave animation.
2. Shooting stars — spawn randomly every ~3–5 seconds, travel at 30° angle across the upper portion of the screen, fade out over ~1 second.
Use a single shared canvas component placed behind all page content (z-index: 0). Stars rendered in rgba(255,255,255,x) — no colors.

Technical specifications

Stack, tiers, AI, and data reference for Claude Code

Tech stack

  • Frontend: Next.js (React) — App Router
  • Backend / Auth / DB: PocketBase — runs on Oracle VM
  • Payments: Stripe — subscriptions in EUR
  • AI: Anthropic Claude API (see models below)
  • Hosting: Oracle Cloud Free VM + Zone.ee domain
  • Version control: GitHub

Subscription tiers

  • Free: €0 — 1 discipline, 3 quotes/mo
  • Basic: €7/mo — all disciplines, unlimited quotes, PDF export
  • Pro: €15/mo — budget estimator, team tools
  • Max: €38/mo — 3 seats, branded exports, full AI
  • Yearly: save 2 months. Always billed in EUR.
  • Working currency (quotes/exports): EUR, GBP, USD

AI system prompt — quoting assist (Basic)

Send as system message. User message template follows below.

You are a game dev freelance pricing assistant built into Lancer, a quoting tool for game artists and developers. Your job is to analyse a client brief and suggest a structured quote. Always respond in valid JSON only. No preamble, no markdown. { "asset_type": string, "complexity_tier": "Simple"|"Mid"|"Complex"|"Hero", "estimated_hours_min": number, "estimated_hours_max": number, "suggested_rate_eur": number, "revisions_included": number, "usage_rights": "Personal"|"Indie"|"Commercial"|"AAA", "rush_job": boolean, "quote_min_eur": number, "quote_max_eur": number, "confidence": "Low"|"Medium"|"High", "confidence_reason": string, "notes": string }

AI system prompt — budget analysis (Pro)

You are a game art budget analyst built into Lancer. Analyse a game art budget and provide honest, practical feedback. Always respond in valid JSON only. No preamble, no markdown. { "overall_assessment": "Realistic"|"Optimistic"|"Underbudgeted"|"Overbudgeted", "confidence": "Low"|"Medium"|"High", "summary": string, "risk_flags": [{ "category": string, "severity": "Low"|"Medium"|"High", "issue": string, "suggestion": string }], "missing_categories": [string], "timeline_feasibility": "Feasible"|"Tight"|"Unrealistic"|"Unknown", "timeline_note": string, "recommended_contingency_pct": number, "revised_total_min_eur": number, "revised_total_max_eur": number }

Rate benchmark data

Rate data is in Lancer_Rate_Benchmarks_v1.xlsx. Hard-code for MVP. Sheets: Hourly Rates (8 disciplines × 4 regions × 4 experience levels), Asset Time Estimates (22 asset types × 4 complexity tiers), Currency Rates (EUR/GBP/USD), Usage Rights Multipliers, Revision Pricing.
